# ESI distribution notes

## Why we care
This paper is not a staffing-optimization paper, but it is useful context.

It provides:
- expected and observed ESI distribution ranges
- a reminder that triage mix varies a lot across hospitals

## Practical takeaway for our project
If we later add patient classes or acuity-based routing, we should use realistic class proportions instead of inventing them blindly.
For the first half, it mainly helps justify why a triage subsystem is not homogeneous in reality.
